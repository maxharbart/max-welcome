# Max Welcome

A unique animated welcome page for GitHub Pages.

Live URL after deployment:

```text
https://maxharbart.github.io/max-welcome/
```

## Deploy

This repository uses GitHub Actions:

```text
.github/workflows/deploy.yml
```

Push to `main`, then open **Actions** and wait for the Pages deploy workflow to finish.

If GitHub asks for Pages source, set:

```text
Settings → Pages → Build and deployment → Source → GitHub Actions
```
